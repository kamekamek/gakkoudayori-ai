## 🎯 主要機能の整理

### 核心機能（現在のベース）
1. **チャットボット形式での情報収集**
   - AIが質問して先生が回答
   - 音声入力・テキスト入力の併用
   - 段階的な情報収集

2. **リアルタイムプレビュー**
   - 会話内容に基づく学級通信の自動生成
   - A4サイズでの印刷用プレビュー
   - 即座の視覚的フィードバック

3. **最小限の設定**
   - 学校名・クラス名の設定
   - ユーザー辞書（音声認識精度向上）
   - 基本的なヘルプ機能

4. **出力機能**
   - PDF生成・ダウンロード
   - 必要に応じた再生成

## 🚀 実装優先度

### Phase 1: 最小限MVP
- **デスクトップ**: 左右分割レイアウト（チャット＋プレビュー）
- **モバイル**: タブ切り替え（チャット⇄プレビュー）
- 基本的なAI会話機能
- シンプルなプレビュー表示
- PDF出力

### Phase 2: 使いやすさ向上
- 音声入力の統合
- レスポンシブ対応の細かな調整
- ユーザー辞書機能

### Phase 3: 細かな改善
- アニメーション・フィードバック
- エラーハンドリング
- パフォーマンス最適化
