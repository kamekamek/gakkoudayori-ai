## 🔧 機能別操作説明

### プレビューモード操作
- **表示**: 読み取り専用での内容確認
- **スクロール**: 長い内容の閲覧
- **ズーム**: ピンチ・ホイールでの拡大縮小

### 編集モード操作
- **テキスト編集**: クリック・タップで直接編集
- **フォーマット**: ツールバーでの装飾
- **画像操作**: ドラッグ&ドロップで挿入・移動
- **自動保存**: 変更時の自動保存

### 印刷ビューモード操作
- **全画面表示**: 実際の印刷レイアウト確認
- **ズーム**: 詳細確認のための拡大・縮小
- **印刷**: ブラウザ印刷機能連携
- **戻る**: 通常のプレビューモードに復帰

### PDF出力機能
- **ワンクリック生成**: 現在の内容でPDF作成
- **自動ダウンロード**: 生成後の即座ダウンロード
- **ファイル名**: 自動生成（学級通信_日付.pdf）

## 🎯 ユーザビリティ配慮

### 直感的操作
- **明確なボタンラベル**: 機能が分かりやすいアイコン+テキスト
- **状態表示**: 現在のモードを明確に表示
- **進行状況**: 処理中の状態をリアルタイム表示

### アクセシビリティ
- **キーボード操作**: 全機能をキーボードで操作可能
- **スクリーンリーダー**: 適切なセマンティクス
- **高コントラスト**: 視認性重視の色設計

### エラー対応
- **自動復旧**: 編集中のデータ保護
- **エラーメッセージ**: 分かりやすい問題説明
- **代替手段**: 機能が使えない場合の別ルート提供
